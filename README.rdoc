== README

Sistema para controle das areas:
    Comercial
          - Contratos de Locação
    Financiera
          - Despesas
          - Receitas
          - Movimentação Financeira
          - Fluxo de Caixa
    Gestão de Clientes
          - Central de Atendimentos
          - Ordem de Serviços
          - Vendas
    Integração com o Google Agenda e Maps

* Ruby  : 2.3
* Rails : 4.2.6 com MySql
